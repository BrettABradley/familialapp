
# Update Favicon

Copy the uploaded Familial logo to the public directory and update `index.html` to use it as the favicon.

## Steps

1. Copy `user-uploads://Familial_PNG_Black_-_watermark-2.png` to `public/favicon.png`
2. Update `index.html` to replace the existing favicon reference with a `<link>` tag pointing to `/favicon.png`
